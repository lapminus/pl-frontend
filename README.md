## Premier League Frontend (PL)

An Angular application for displaying football (soccer) player data.
This project focuses on the frontend, using the Premier League REST API to provide a responsive UI.

## Features

- CRUD operations for player cards
- Bootstrap
- Pagination
- Search functionality
